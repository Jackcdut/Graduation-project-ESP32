◄ 电流读取反馈 (I_fb)

工作原理说明

● CV模式（恒压）

当 I_out < I_set 时，电压环控制

输出电压 = V_set（设定值）

○ CC模式（恒流）

当 I_out ≥ I_set 时，电流环控制

输出电流 = I_set（设定值）

切换原理：二极管或门实现"低选控制"，输出较低电压的环路自动获得控制权，实现无缝平滑切换。
