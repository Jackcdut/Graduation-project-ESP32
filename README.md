# ESP32-P4 毕业设计项目

ESP32-P4 + ESP32-C6 多功能开发板项目，包含LVGL GUI、OneNET云平台和Web仪表盘。

## 📁 版本管理

本仓库采用文件夹方式管理不同版本，每个版本都是一个独立的完整项目。

### 当前版本

- **[V3.0](./V3.0/)** - 最新版本（2026-01-25）
  - ✅ **示波器功能完善** - 全面优化示波器核心功能
  - ✅ **FFT频谱分析** - 支持FFT频谱分析和谐波分析
  - ✅ **数据导出功能** - 支持波形数据和FFT数据导出
  - ✅ **界面美化** - 示波器相关页面进一步美化优化
  - ✅ **ESP32-P4内置ADC测试** - 使用ESP32-P4内置ADC作为采样源进行波形显示测试
  - 基于V2.0的所有功能

- **[V2.0](./V2.0/)** - 稳定版本（2026-01-18）
  - ✅ **无线串口功能完善** - 现在能够正常使用
  - ✅ **视频播放功能** - 支持AVI视频播放（复杂画面会有轻微卡顿）
  - 基于V1.0的所有功能

- **[V1.0](./V1.0/)** - 首个正式版本
  - ESP32-P4 + ESP32-C6 双芯片架构
  - LVGL v8.3 全屏双缓冲显示
  - 网络模块（WiFi、OneNET、天气API）
  - 11个GUI界面屏幕
  - 示波器、无线串口、截图功能
  - Web数据可视化仪表盘
  - 性能优化（100 FPS理论帧率）

## 🔧 编译和烧录

### 主程序 (ESP32-P4)
```bash
cd V3.0
idf.py build
idf.py -p COM3 flash monitor
```

### 从机程序 (ESP32-C6)
```bash
cd V3.0
idf.py -B build_slave build
idf.py -B build_slave -p COM4 flash
```

## ✨ V3.0 主要改进

### 示波器功能增强
- **FFT频谱分析**: 实时FFT变换，支持频谱显示和谐波分析
- **数据导出**: 支持波形数据和FFT频谱数据导出为CSV格式
- **测量优化**: 改进波形测量算法，提高精度
- **界面美化**: 优化示波器UI设计，提升用户体验
- **ADC集成**: 使用ESP32-P4内置ADC进行实际波形采样测试

### 技术亮点
- 高性能FFT算法实现
- 优化的数据采集和处理流程
- 完善的导出功能支持数据分析
- 模块化代码架构，易于维护和扩展
